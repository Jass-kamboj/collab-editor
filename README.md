# Collab Editor

A real time collaborative document editor built with JavaFX and WebSocket.

## Features
- Real time collaboration between multiple users
- Text editing with live sync
- WebSocket server and client
- Coming soon: Bold, italic, tables, images, save to PDF and DOCX

## How to run

### Requirements
- JDK 25
- Maven

### Start the server
```bash
mvn exec:java "-Dexec.mainClass=com.example.EditorServer"
```

### Start the editor
```bash
mvn javafx:run
```
